# Fase 56 - Dashboard Operacional Sintético de Alimentador

## Objetivo

A Fase 56 adiciona um painel web para demonstrar o agente sintético de perdas em uma visão operacional de alimentador.

O painel foi inspirado em relatórios técnicos de alimentador, com foco em subestação, percurso, religadores, clientes MT, transformadores, GD, perdas e criticidade operacional.
