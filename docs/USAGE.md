# Guia de Uso

## Objetivo

Este documento descreve como executar o sistema Area 51 - Aerial Housing Detection em ambiente local, pela API e por scripts operacionais.

O sistema recebe imagens aéreas, executa o pipeline de detecção de candidatos a telhado, estima a quantidade de residências e gera saídas em CSV e HTML.
